# House Price Prediction using Linear Regression
SkillCraft Technology Internship - Task 01
## Overview

This project implements a Linear Regression model to predict house prices based on:

* Square Footage (GrLivArea)
* Number of Bedrooms (BedroomAbvGr)
* Number of Bathrooms (FullBath)

The project uses the House Prices dataset and applies Machine Learning techniques to estimate house prices.

## Dataset

Dataset: House Prices - Advanced Regression Techniques

Features Used:

* GrLivArea (Above Ground Living Area)
* BedroomAbvGr (Number of Bedrooms)
* FullBath (Number of Bathrooms)

Target Variable:

* SalePrice

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-learn
* Google Colab

## Machine Learning Model

* Linear Regression

## Steps Performed

1. Loaded the dataset.
2. Selected relevant features.
3. Split the dataset into training and testing sets.
4. Trained a Linear Regression model.
5. Predicted house prices on test data.
6. Evaluated model performance.
7. Visualized Actual vs Predicted Prices.

## Results

MAE (Mean Absolute Error): 35788.06

MSE (Mean Squared Error): 2806426667.25

R² Score: 0.6341

## Visualization

A scatter plot was generated to compare Actual House Prices and Predicted House Prices.

## Conclusion

The Linear Regression model successfully predicted house prices using square footage, number of bedrooms, and number of bathrooms. The model achieved an R² Score of 0.6341, indicating a reasonable prediction performance for the selected features.
